# Huffman in C

## About 

This program is encoder and decoder for compressing.

## Usage 

Compile

``````````````````````````````````````````````````````````````````````````````````typescript
$ gcc -o huffman huffman.c
``````````````````````````````````````````````````````````````````````````````````

Encode 

```
$ ./huffman encode <input file> <output file>
```

Decode

```
$ ./huffman decode <input file> <output file>
```
